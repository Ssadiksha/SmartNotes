# SmartNotes 📝

A clean and organized notes application built with **Node.js**, **Express.js**, and **MySQL**.  
SmartNotes allows users to create, read, update, and delete notes using a simple REST API backend and a lightweight frontend interface.

---

## Features

- Create new notes
- View all notes
- Update existing notes
- Delete notes
- MySQL database integration
- RESTful API design
- Clean and scalable project structure
- Environment variable support with `.env`
